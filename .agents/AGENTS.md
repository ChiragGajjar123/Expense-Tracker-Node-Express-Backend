# Agent Instructions

Welcome to the **Expense Tracker Node Express Backend** repository. This document outlines the workspace guidelines and rules for AI agents collaborating on this project.

## Technology Stack & Architecture
- **Runtime**: Node.js
- **Framework**: Express.js
- **Code Style**: Clean, modular, and well-documented JavaScript/TypeScript.
- **Security**: Always sanitize inputs, use environment variables for secrets, and implement CORS/Helmet.

## Instructions for Agents
- Propose clean and minimal changes.
- Use `async`/`await` for asynchronous code block handling.
- Ensure proper error handling and logging in all controllers and middleware.
- Do NOT push code to the remote repository without explicit user permission in any scenario.
